### Features
- Localization (English , Arabic , Soon more will be added)
- Reciter Control (Choose default reciter - change current surah reciter)
- Recitations
- Recommendations
- Sleep Mode
- Reading
- Share Ayah 
- Offline Mode (download and be able to listen offline)
